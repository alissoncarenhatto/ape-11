COMANDOS GIT FLOW

Comando Descrição
git config --list Para verificar as configurações locais
git config --global user.name Para encontrar o nome de usuário
git config --global user.email Para encontrar o email
git config --global user.name "nome do usuário" Alterar o nome de usuário
git config --global user.email "email do usuário" Alterar o email
git init Iniciar um repositório
git clone link “Baixar” um repositório
git pull Baixar as últimas alterações do servidor
git remote -v Listando o caminho do servidor
git remote set-url origin git://url Adicionando o caminho do servidor
git add nome_do_arquivo Adicionando um arquivo
git add . Adicionando tudo de uma vez
git reset nome_do_arquivo Removendo arquivos
git commit -m "mensagem" Pacote de alterações feitas no repositório
git status Verificando o que foi alterado
git branch Listando as branches existentes
git branch nome Criando uma nova branch
git checkout -b nome_da_nova_branch Criando uma nova branch e já trocando para ela
git branch -d nome Deletando uma branch
git checkout nome_da_branch_existente Trocando de branch
git push origin nome_da_branch Enviando uma branch para o servidor
git push origin :nome_da_branch Deletando uma branch remota
git tag Listando as tags Git
git tag 0.0.1 Criar uma tag Git
git tag -a 0.0.1 -m "versão 0.0.1" Criar uma tag com mensagem (anotada)
git tag -a 0.0.1 (hash commit) Criar uma tag a partir de um commit
git push origin 0.0.1 Criando a tag no servidor
