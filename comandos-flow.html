COMANDOS GIT FLOW

Comando Descrição
git flow feature start [NOMEDORECURSOQUEVOCÊQUERCRIAR] Cria um branch feature/[NOMEDORECURSOQUEVOCÊQUERCRIAR] para o desenvolvimento de novos recursos
git flow feature finish [NOMEDORECURSOQUEVOCÊQUERCRIAR] Finaliza o branch feature/[NOMEDORECURSOQUEVOCÊQUERCRIAR] unindo o mesmo com o branch develop.
git flow release start [versão] Cria um branch release/[VERSAO] para preparar o código para unir com o branch master.
git flow release finish [versão] Finaliza o branch release/[VERSAO] unindo o mesmo com o branch master.
git flow hotfix start [versão] Cria um branch hotfix/[VERSAO] para a correção de bugs críticos.
git flow hotfix finish [versão] Finaliza o branch hotfix/[VERSAO] unindo o mesmo com o branch master.
